About Me.

SOFTWARE ENGINEER & WEB DEVELOPER STUDENT

Saya Fabian Rizky Pratama, seorang siswa software engineer dan web developer yang memiliki ketertarikan besar di bidang teknologi. Saat ini saya duduk di kelas XI tahun ajaran 2025/2026 dan menempuh pendidikan jurusan Rekayasa Perangkat Lunak di SMK Budi Luhur.

Saya berfokus pada pengembangan frontend dan backend, dengan tujuan menciptakan aplikasi yang modern, efisien, dan ramah pengguna. Dengan dasar yang kuat dalam pemrograman dan pemecahan masalah, saya senang mengubah ide menjadi solusi digital yang fungsional dan menarik secara visual.

Saya selalu antusias untuk mempelajari teknologi baru, mengerjakan proyek-proyek menarik, serta terus mengembangkan kemampuan saya di bidang pengembangan web dan perangkat lunak.
